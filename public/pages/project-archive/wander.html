Wonder, Wander, Repeat

Python app

Graphical User Interface

Mapping

Wonder, Wander, Repeat is an application developed as an assistance for planning travels according to user preferences such as temperature, precipitation and landforms.

The GUI of application is developed using Tkinter widgets in python. Plotting of maps of USA for depicting information is done by using plotly library graph objects Choropleth. The maps show relevant information on mouse hover. The data to be plotted by these maps is provided in the form of csv files. Panda library is used for reading information from csv files.

Pandas is a Python package providing fast, flexible, and expressive data structures designed to make working with structured and time series data both easy and intuitive. Pandas is the most popular python library that is used for data analysis.

Tkinter is the standard GUI library for Python. Python when combined with Tkinter provides a fast and easy way to create GUI applications. Tkinter provides a powerful object-oriented interface to the Tk GUI toolkit.

The project combines multiple python libraries and stores user data in a mySQL database. Some key features include making a user profile, travel wishlist, landform, precipitation and temeperature mapping of different places as well as places to visit in the chosen destination.
